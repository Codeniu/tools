# 图片拼接工具 - Image Merger

简单高效的图片拼接工具，将多张图片拼接成一张大图，支持水平和垂直两种拼接方式。

## 简介

图片拼接工具是一个完全运行在浏览器中的在线工具，可以将多张图片按照指定的方向拼接成一张大图。所有处理都在本地完成，无需上传到服务器，确保您的图片隐私安全。支持长图制作、对比图制作等多种场景。

## 主要功能

### 核心特性

- ✅ **两种拼接模式** - 支持水平拼接（垂直排列）和垂直拼接（水平排列）
- ✅ **批量上传** - 支持一次选择多张图片
- ✅ **拖拽上传** - 支持点击上传和拖拽上传两种方式
- ✅ **顺序调整** - 可以通过上移/下移按钮调整图片拼接顺序
- ✅ **间距设置** - 自定义图片之间的间距（0-100像素）
- ✅ **背景颜色** - 自定义背景颜色，支持颜色选择器
- ✅ **多格式输出** - 支持 PNG、JPEG、WebP 三种输出格式
- ✅ **实时预览** - 拼接完成后立即预览结果
- ✅ **完全本地** - 所有处理在浏览器本地完成，保护隐私

### 拼接模式详解

#### 水平拼接（Horizontal）
- 图片从上到下依次排列
- 自动居中对齐
- 适合制作长截图、教程图等
- 所有图片以最大宽度为基准

#### 垂直拼接（Vertical）
- 图片从左到右依次排列
- 自动居中对齐
- 适合制作对比图、全景图等
- 所有图片以最大高度为基准

## 使用说明

### 基本使用步骤

1. **上传图片**
   - 点击上传区域选择图片
   - 或直接拖拽图片到上传区域
   - 支持一次选择多张图片
   - 支持格式：JPG、PNG、GIF、WebP 等

2. **调整图片顺序**
   - 使用"↑ 上移"按钮将图片向上移动
   - 使用"↓ 下移"按钮将图片向下移动
   - 使用"删除"按钮移除不需要的图片
   - 图片序号显示拼接顺序

3. **设置拼接参数**
   - **拼接方向**：选择水平或垂直拼接
   - **图片间距**：设置图片之间的间距（0-100像素）
   - **背景颜色**：点击颜色选择器选择背景色
   - **输出格式**：选择 PNG、JPEG 或 WebP

4. **开始拼接**
   - 点击"开始拼接"按钮
   - 等待处理完成
   - 查看拼接结果

5. **下载结果**
   - 点击"下载结果"按钮
   - 选择保存位置
   - 完成下载

## 功能限制与注意事项

### 当前限制
- 建议单次拼接不超过 20 张图片
- 建议单张图片大小不超过 20MB
- 支持的图片格式：JPG、PNG、GIF、WebP 等

### 使用建议
1. **图片质量** - 高质量图片拼接效果更好
2. **拼接数量** - 避免过多图片拼接，可能影响性能
3. **格式选择**：
   - 需要透明背景选择 PNG
   - 照片类图片选择 JPEG 或 WebP
   - 追求小体积选择 WebP
4. **间距设置**：
   - 长截图建议 0 间距
   - 对比图建议 10-20 间距
   - 教程图建议 5-10 间距
5. **背景颜色**：
   - 长截图建议白色或透明
   - 对比图建议浅色背景
   - 艺术创作可自由选择

### 常见问题

**Q1: 为什么拼接后的图片有白边？**
A: 这是由于背景颜色设置导致的。如需透明背景，请确保选择 PNG 格式，并在有间距时注意背景色的影响。

**Q2: 支持拼接不同尺寸的图片吗？**
A: 完全支持。工具会自动将图片居中对齐，保持最佳视觉效果。

**Q3: 可以调整已上传图片的顺序吗？**
A: 可以。使用每张图片卡片上的"↑ 上移"和"↓ 下移"按钮调整顺序。

**Q4: 拼接后的图片质量会下降吗？**
A:
- PNG 格式：不会下降（无损）
- JPEG/WebP 格式：可能有轻微损失（质量设为 90%）

**Q5: 可以上传多少张图片？**
A: 理论上无限制，但建议单次不超过 20 张以获得最佳性能。

**Q6: 拼接完成后可以再调整顺序吗？**
A: 可以。调整顺序后需要重新点击"开始拼接"生成新的结果。

**Q7: 支持移动设备吗？**
A: 完全支持。采用响应式设计，在手机、平板、电脑上都能正常使用。
