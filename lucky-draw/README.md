# 在线抽奖工具 - Lucky Draw

简单、公平、有趣的在线抽奖系统，纯前端实现，保护隐私。

## 简介

这是一个完全运行在浏览器中的抽奖工具，无需安装任何软件或插件。采用完全随机的抽取算法，确保抽奖的公平性。所有数据均存储在本地，不会上传到任何服务器，充分保护用户隐私。

## 主要功能

### 核心特性

- ✅ **灵活的名单管理** - 支持批量输入参与者，每行一个名字
- ✅ **可自定义设置** - 中奖人数、抽奖速度、抽奖时长均可调节
- ✅ **公平随机抽取** - 完全随机算法，已中奖人员不会重复中奖
- ✅ **丰富视觉效果** - 数字滚动动画、庆祝彩带效果
- ✅ **数据持久化** - 使用 LocalStorage 保存数据，刷新不丢失
- ✅ **结果导出** - 支持导出中奖名单为 TXT 文件
- ✅ **实时统计** - 显示总参与人数、已抽取人数、剩余人数

### 抽奖设置

- **中奖人数**：可设置一次抽取 1 到多人
- **抽奖速度**：50-500 毫秒可调，控制数字滚动速度
- **抽奖时长**：1-10 秒可调，控制抽奖持续时间

### 视觉反馈

- 渐变动画背景效果
- 数字快速滚动动画
- 中奖时彩带庆祝动画
- 中奖记录滑入动画
- 实时状态提示

## 使用说明

### 基本使用步骤

1. **输入参与名单**
   - 在文本框中输入参与抽奖的人员名单
   - 每行输入一个名字
   - 支持复制粘贴 from Excel 或其他文档

   示例：
   ```
   张三
   李四
   王五
   赵六
   ```

2. **设置抽奖参数**
   - **中奖人数**：设置一次抽取的中奖人数（默认 1 人）
   - **抽奖速度**：设置数字滚动速度，数值越小越快（默认 100ms）
   - **抽奖时长**：设置抽奖持续时间（默认 3 秒）

3. **开始抽奖**
   - 点击"开始抽奖"按钮
   - 观察数字滚动动画
   - 等待抽奖结果
   - 查看中奖名单

4. **管理结果**
   - 中奖结果会自动显示在中奖名单区域
   - 包含中奖序号和抽取时间
   - 点击"导出结果"可下载 TXT 文件
   - 点击"重置"可清空所有中奖记录

### 使用技巧

1. **批量导入名单**
   - 从 Excel 复制一列名字，直接粘贴到文本框
   - 从文本文件复制名单，每行一个名字

2. **多次抽奖场景**
   - 工具会自动记录已中奖人员
   - 已中奖人员不会再次被抽中
   - 适合分批抽取多个奖项的场景

3. **数据保存**
   - 所有数据自动保存在浏览器本地
   - 关闭或刷新页面后数据不会丢失
   - 更换浏览器或清除缓存会丢失数据

4. **导出功能**
   - 点击"导出结果"按钮
   - 自动下载包含所有中奖名单的 TXT 文件
   - 文件名包含日期信息

## 常见问题

### Q1: 为什么我的名单输入后不显示？
A: 请确保每行只有一个名字，多余的空行会被自动过滤。检查名字是否包含特殊字符。

### Q2: 已中奖的人还会再次被抽中吗？
A: 不会。系统会自动记录已中奖人员，确保不会重复中奖。如需重新开始，请点击"重置"按钮。

### Q3: 数据会保存多久？
A: 数据保存在浏览器本地，除非清除浏览器缓存或数据，否则会一直保存。

### Q4: 可以导出中奖结果吗？
A: 可以。点击"导出结果"按钮，会自动下载包含所有中奖信息的 TXT 文件。

### Q5: 抽奖结果公平吗？
A: 完全公平。使用 JavaScript 的 Math.random() 函数生成随机数，这是业界标准的真随机算法。

### Q6: 支持多少人同时参与？
A: 理论上无限制。但在实际使用中，建议单次抽奖不超过 1000 人，以获得最佳性能。

### Q7: 可以在手机上使用吗？
A: 完全可以。采用响应式设计，在手机、平板、电脑上都能完美使用。
