# PNG 转 ICO 在线转换器

这是一个完整的 HTML 页面，实现了一个功能丰富的 PNG 转 ICO 在线转换器。

## 核心功能

- **图片上传**：支持点击选择和拖拽上传，支持 PNG、JPG、JPEG、BMP 格式
- **多尺寸选择**：提供 16x16 到 256x256 的多种图标尺寸选项
- **实时预览**：选择图片后可预览不同尺寸的转换效果
- **批量转换**：支持单尺寸或多尺寸同时转换
- **智能下载**：单尺寸直接下载 ICO 文件，多尺寸打包成 ZIP 下载

## 技术特点

- **纯前端实现**：完全在浏览器中处理，无需服务器
- **ICO 格式规范**：严格按照 ICO 文件格式生成，包含 DIB 头和 AND mask
- **响应式设计**：适配移动端和桌面端
- **优雅的界面**：渐变色主题，现代化 UI 设计

## 代码结构

- **HTML 结构**：语义化标签，清晰的内容分区
- **CSS 样式**：使用现代 CSS 特性，包含动画效果和媒体查询
- **JavaScript 实现**：面向对象设计，使用 ES6+ 语法，包含完整的 ICO 文件生成算法

## 文件说明

这个 `png-to-ico.html` 文件是一个独立运行的网页应用，可以直接在浏览器中打开使用，为用户提供便捷的图标格式转换服务。

### 使用方法

1. 在浏览器中打开 `png-to-ico.html` 文件
2. 点击上传区域或拖拽图片文件到指定区域
3. 选择需要的图标尺寸
4. 点击"转换为 ICO"按钮
5. 等待转换完成并下载生成的文件

### 支持的格式

- 输入格式：PNG, JPG, JPEG, BMP
- 输出格式：ICO（单文件）或 ZIP（多文件批量）
